

BSE EQUITY SEARCH App that downloads data from bse website and updates in redis,served from django backend and displayed using vue frontend




## Installation

Backend

```sh
cd backend
pip3 install -r requirements.txt
python3 manage.py runserver
```

Frontend

```sh

cd frontend
yarn install
yarn serve

```
